# Game API
# Project structure
- Docker
- MariaDB
- Node (24.12.0)

# Instructions

## Install depepdencies
- Run `npm install`
- Run `docker-compose up`

## Api documentation
- Run `npm run docs`
- Run `open apidoc/index.html`

## Testing
- supertest (in proggress)
- postman (Prefered way)
- folder postman contains ready to use api calls for this project as Collection
